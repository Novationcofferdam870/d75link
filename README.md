## 🧭 Overview

d75link is a Windows app for the Kenwood TH-D75 radio. It helps you connect through Bluetooth and use D-STAR reflectors with a simple terminal-style screen.

Use it to link to REF, XRF, and DCS reflectors from your radio setup. The app keeps the process in one place, so you can focus on radio use instead of menu steps.

## ✅ What You Need

Before you start, make sure you have:

- A Windows PC
- A Kenwood TH-D75 radio
- Bluetooth on your PC
- The radio paired with your PC
- Internet access for the first download

A desktop or laptop with Windows 10 or Windows 11 works best. The app uses a lightweight interface and does not need a large amount of system memory.

## 📡 How to Use d75link

Once the app is open:

1. Turn on Bluetooth on your PC.
2. Pair your TH-D75 with the computer.
3. Start d75link.
4. Select or enter the reflector you want to use.
5. Connect to REF, XRF, or DCS as needed.
6. Use the radio as you normally would.

The terminal-style interface keeps the layout plain and easy to read. You can watch the connection state and reflector target from one screen.

## 🔧 Basic Troubleshooting

If the app does not start:

- Download the file again
- Make sure the download finished
- Extract the ZIP file if you downloaded one
- Run the app from the extracted folder
- Check that Bluetooth is on
- Re-pair the radio and PC if needed

If the radio does not connect:

- Move the PC closer to the radio
- Turn Bluetooth off and on again
- Remove the old pairing and add it again
- Restart the app
- Restart the radio

If the reflector does not link:

- Check the reflector name
- Make sure you typed the right REF, XRF, or DCS target
- Try again after reconnecting the radio
- Confirm that the radio has a live Bluetooth link

## 🔌 Bluetooth Pairing Tips

To pair the TH-D75 with Windows:

1. Open Windows Settings.
2. Go to Bluetooth and devices.
3. Turn Bluetooth on.
4. Put the radio in pairing mode.
5. Add the radio from the Windows device list.
6. Confirm the pairing code if Windows shows one.

After pairing, open d75link and test the connection. If the pair stops working later, remove the device from Windows and pair it again.
